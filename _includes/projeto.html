Esta é uma proposta para a criação de um laboratório de pesquisas de novas naturezas. Especificamente para o desenvolvimento de uma astronomia híbrida (natural + artificial) e uma cartografia eletromagnética.

O projeto consiste no desenvolvimento de tecnologias de captação, processamento e armazenamento de sinais/ondas/dados eletromagnéticos. Os resultados se materializarão em forma de: workshop, publicação, exposição e a construção de uma plataforma online para o compartilhamento de dados abertos sobre os sinais captados.
CAPTAR
Criação e experimentação com antenas socionaturais. Essas são antenas construídas a partir de materiais naturais (plantas, cristais) e processados (metais, resinas), seus emaranhamentos e significações contemporâneas.
Caracterização
: Após construídas as antenas precisam ser testadas para caracterizar suas propriedades físicas como: frequências de reverberação e sensitividade espacial.
PROCESSAR
Construção de sistemas para filtrar, organizar e tentar fazer sentido de dados provindo de múltiplas antenas e fatias do espectro eletromagnético. Técnicas e algoritmos de Machine Learning podem ser usadas para explorar padrões em tempo real e longo prazo.
Visualização
: Sistemas para facilitar a leitura e análise desses sinais e seus eventuais padrões para explorar como podemos evitar o erro antropocêntrico de criar epistemologias usando somente os arbitrários sinais linguísticos humanos.
ARMAZENAR
Estratégias para o armazenamento e tagueamento de dados, e a programação de uma plataforma online com uma API para facilitar o acesso a esses dados.
Compartilhamento
: Dados coletados poderão ser usados em análises que, por exemplo, cruzam dados econômicos sobre uma região com a presença de certos sinais eletromagnéticos.
